Split & Play

AI-powered practice companion for musicians

Upload a song, AI separates it into 7 stems (vocals, drums, bass, guitar, piano, other, click track), then practice with a real-time mixer, tempo/pitch control, and intelligent beat detection.

Problem: Musicians need to isolate instruments to learn parts, but existing tools are expensive or limited to basic separation.

Next.js TypeScript Supabase Replicate Howler.js Tone.js Modal
Key implementation details
  • 7-stem separation using Demucs AI model via Replicate
  • Smart click track generated via Modal endpoint with librosa beat detection
  • Real-time audio engine: tempo 0.25x–2x, pitch ±12 semitones
  • A-B looping, section markers, session tracking with CSV export
  • Fire-and-forget architecture with frontend polling for Vercel compatibility

02

PromptLens

Visual prompt engineering from design screenshots

Upload design screenshots, Claude Vision extracts styles (colours, typography, spacing, layout) into structured data, then generates AI-ready prompts optimised for Cursor, Stitch, and Anima.

Problem: Developers spend 30+ minutes manually translating designs into code prompts. PromptLens does it in seconds with computer vision.

Next.js TypeScript Supabase Claude Vision API Stripe
Key implementation details
  • Claude Vision analyses screenshots as base64 (private URLs can't be fetched externally)
  • Structured JSON extraction + natural language prompt in single API call
  • Multi-tool export: Cursor, Stitch, Anima, Generic — each optimised for the target
  • Public prompt library for community sharing
  • Stripe billing with server-side usage enforcement

03

Insight

RAG document search with vector embeddings

Upload documents (PDF, TXT, MD), they're chunked and embedded with OpenAI text-embedding-3-small, stored in Supabase pgvector, then searched via cosine similarity to ground AI responses in your actual content.

Problem: Generic AI chat hallucinates. Insight constrains answers to your uploaded documents with source citations.

Next.js TypeScript Supabase pgvector OpenAI Vercel AI SDK
Key implementation details
  • Full RAG pipeline: chunk → embed → store → retrieve → generate
  • pgvector IVFFlat index with cosine distance search
  • Streaming responses with [Source N] citation notation
  • Expandable source cards showing matched passages and similarity scores
  • Intelligent chunking: 500 tokens with 50-token overlap on paragraph boundaries

How I use AI

Beyond API calls and chatbot wrappers. I build AI into product infrastructure — from vector search to multi-agent systems to computer vision pipelines.

Multi-agent orchestration

I run 6 specialised AI agents on a Mac Mini M4 Pro — each handles a different domain (research, design, marketing, development, client services). They share context, delegate tasks to each other, and run scheduled jobs autonomously. Not a toy setup: these agents deploy code, write database migrations, and generate production content.

OpenClaw Sonnet Opus Cron scheduling

RAG architecture

Built a full retrieval-augmented generation pipeline for Insight: document ingestion, intelligent chunking with paragraph-boundary splitting, vector embeddings via OpenAI text-embedding-3-small, pgvector cosine similarity search, and grounded AI responses with source citations. Understands when to say "I don't know" instead of hallucinating.

pgvector OpenAI embeddings Cosine similarity Streaming

Computer vision extraction

PromptLens sends design screenshots to Claude Vision as base64 (Supabase signed URLs can't be fetched externally), extracts structured style data (colours, typography, spacing, borders, shadows, layout patterns), and generates tool-specific prompts. One API call, structured JSON output plus natural language prompt.

Claude Vision Base64 Structured extraction

AI content pipelines

Generated 1,425 curriculum-aligned exam questions using a 3-step pipeline: Ollama generates base questions locally (free), then generates SEND-adapted variants for 4 learning profiles, then Claude reviews every question for accuracy. Each question verified against the National Curriculum. Total API cost: under £5.
